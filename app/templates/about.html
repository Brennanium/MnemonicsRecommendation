Mnemonic Recommendation System

About

The purpose of the Mnemonic Recommendation System is to help users learn new languages by providing them with a mnemonic in order to memorize and learn new words. Currently, our system takes in a user supplied foreign language word, and produces mnemonics for aid in memorizing the supplied word. These mnemonics are similar to the input word semantically, orthographically, phonetically, and also are similar in imageability. Our system is also capable of using the generated mnemonics, or manually created mnemonics by the user, and building a sentence out of the mnemonic, for additional aid in memorization. The system utilizes OpenAI-GPT and DistilBERT techniques for building a sentence around an input mnemonic. The system can be repeatedly prompted for new sentences to find one that is helpful in memorization.
